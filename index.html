What is miPereZ?

miPreZ

miPreZ is a minimal HTML slides framework to make slides easy to edit inline and online.

How to use?

Just copy the HTML code of this presentation and edit away. The simplicity of it allows me to just git clone the repo directly into my server and then just edit away. if you want a code block to be rendered from markdown just add the **md** class to it.

Are there any Themes?

Right now there are three themes light (default), dark and grad. Just change the class in the body tag.
